前言

在有些场景中，我们需要在容器内，获取某个服务器上的多个文件。我们该如何获取呢？

正常的步骤是先将文件上传到容器所在服务器，再将文件复制到容器中。这样做当然可以，但如果每个文件都这样，就会显得十分繁琐（当然，其他获取文件的场景也适用）。

这里有个小技巧推荐给大家：将该目录转变成一个文件http服务，向外提供下载。
